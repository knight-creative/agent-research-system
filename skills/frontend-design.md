# Frontend Design Skill Summary

This skill guides the creation of distinctive, production-grade web interfaces that prioritize authentic design over generic aesthetics.

## Core Approach

Before implementing, designers should clarify the interface's purpose, target audience, and commit to a specific aesthetic direction—whether brutalist, maximalist, minimalist, or another intentional style.

## Key Implementation Principles

**Typography**: Select characterful, unexpected fonts rather than defaults. "Pair a distinctive display font with a refined body font" to elevate visual interest.

**Color & Composition**: Commit to cohesive palettes with dominant colors and sharp accents. Use asymmetry, overlap, and grid-breaking elements for spatial impact.

**Motion & Interaction**: Prioritize high-impact animations during page load with staggered reveals and scroll-triggered effects rather than scattered micro-interactions.

**Visual Depth**: Build atmosphere through gradients, textures, patterns, and layered effects that match the aesthetic direction.

## What to Avoid

The skill explicitly cautions against generic patterns: overused typefaces, predictable layouts, clichéd color schemes (particularly purple gradients), and cookie-cutter components that lack contextual character.

The overarching message emphasizes that **intentionality matters more than intensity**—both refined minimalism and bold maximalism succeed when executed with precision and purpose.
